## Responsive Design Notes

### Mobile (< 768px)
- Stack form fields vertically
- Full-width buttons
- Smaller chart components
- Simplified token table (show less info)
- QR code larger for easy scanning

### Tablet (768px - 1024px)
- 2-column layouts
- Medium-sized charts
- Side-by-side comparisons

### Desktop (> 1024px)
- Full details visible
- Large charts with multiple indicators
- Detailed token tables
- Side panels for additional info
